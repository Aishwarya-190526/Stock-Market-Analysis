## Project Overview
This project focuses on analyzing stock market data to understand price movements, returns, and overall market performance. The dashboard provides insights into historical trends, trading activity, and stock behavior to support data-driven investment analysis.

## Objectives
- Analyze historical stock price trends
- Evaluate stock returns and volatility
- Compare performance across companies and sectors
- Track market behavior over time

## Tools & Technologies
- Excel
- Power BI
- Tableau
- SQL

## Key Insights
- Stock price and return trend analysis
- Volatility assessment across different stocks
- Sector-wise performance comparison
- Identification of top and underperforming stocks


## Conclusion
This dashboard demonstrates practical stock market analysis using Power BI and Tableau, supported by data preparation and querying with Excel and SQL. It highlights the ability to convert complex financial data into clear and actionable insights.
